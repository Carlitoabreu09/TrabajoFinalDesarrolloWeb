##Documentacion



El archivo de configuracion del servidor se llama server.js.

Dentro de la carpeta public se encuentran todos los archivos css,html y javascript del proyecto,
para iniciar el servidor solo hay que moverse al mismo directorio del archivo server.js y escribir en la terminal

**npm run dev**

El servidor ahora se encuentra funcionando y cumpliendo su funcion perfectamente.

En el servidor se instalo la libreria nodemon para automatizar el refresco del servidor
a la hora de hacerle algun cambio y asi no tener que refrescar la pagina.
